Hematology

Hematology here is not a course and not a checklist. It is a reference system built around mechanisms, constraints, patterns, and interfaces — designed for rereading and returning when something doesn’t click.

Concept-first Non-procedural Non-evaluative Calm by design
Concept Boundary

This section focuses on how hematology systems behave and why CBC patterns form. It does not provide disease catalogs, diagnostic algorithms, treatments, instrument menus, or step-by-step workflows.

If a topic cannot stop cleanly at explanation (without turning into “what to do”), it does not belong here.

Orientation

How to use this section

You do not need to read in order. Start with confusion. Follow the links that stabilize the missing mental model.

Start anywhere

Use a Tier if you know the type of problem. Use search if you only remember a term.

Return often

These pages are written to be reread. Understanding deepens through repetition without pressure.

Instructor Note

The goal is explanatory coherence: being able to describe a result in terms of mechanisms and constraints, not just labels.

Common Trap

When hematology feels confusing, it’s often because a stable-looking CBC is being treated like a complete inventory. These deep dives exist to stabilize the missing structure — not to add more facts.

Normal range is a population boundary, not a guarantee of system health.
